MyStudyLife

Web App Development

A student attendance register system, the web app allows a student to log their class attendance via a GPS enabled smartphone. The student turns on GPS/Location on their device, logs in, and selects "Attend" for the module being lectured that they present at. The system checks the student's Device GPS Coordinate against the assigned Vanues's GPS coordinates and checks the student's Device GPS Coordinate ping time against the Subject's assigned attendance time then processes the attendance. This project is developed with Laravel, Bootstrap, HTML, CSS, and it makes use of the MySQL database. The source code is available for review on my Github.
